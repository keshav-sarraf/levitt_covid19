When will it all end

This is a rudimentary model to predict rough estimates of when we can expect Covid19 cases in India to reach lower levels. This model assumes that all efforts that are being undertaken by citizens and the government would continue to remain same, any lapse in safety precautions could definitely lead to escalation of the situation. Please note this is for mere demonstration purposes and there is no guarantee that world would resume to pre Covid19 times as predicted by the model

Levitt Score is a very crude way of measuring the spread of pandemic, it is determined as a ratio cumulative number of cases observed till today with respect to the cumulative number of cases till yesterday. If there were no new additions today, then the ratio would become 1. What this app does is that it tries to predict using linear regression that when can we expect this metric to reach the target of 1 following the current trend.

We can try fitting a curve to the data to predict when the curve would hit the score of 1
